Detailed Description

The CyphalUdp struct is a network node. It is used to send and recieve Cyphal messages over Cyphal/UDP protocol.

To use it, you should instantiate it and set base network address to it:

cyphalpp::CyphalUdp node(socketsImpl, timersImpl);
node.setAddr(someAddr);
Examples:
basic_qt_publisher/main.cpp.
